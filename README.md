# vue-cli3

基于vue-cli3搭建的vue项目，包含内容
1.vue-router
2.axios本地跨域调试以及接口模块化
3.引用cdn资源
4.页面进度条
5.缓存页面配置

## 安装项目
```
yarn install
```

### 本地热更新模式
```
yarn run serve
```

### 生产模式
```
yarn run build
```

### 执行test
```
yarn run test
```

### eslint
```
yarn run lint
```

### Customize configuration
See [Configuration Reference](https://cli.vuejs.org/config/).
